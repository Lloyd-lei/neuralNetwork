Conclusion

This experiment successfully demonstrates the performance comparison between MLP and CNN models on the MNIST dataset. Here are the key findings:

  1. Accuracy: The CNN model achieves slightly higher test accuracy (98.83%) compared to the MLP model (98.10%), confirming CNN's advantage in image classification tasks.
  2. Training Time: The MLP model trains significantly faster (35.37 seconds) than the CNN model (83.70 seconds), making it more efficient for rapid prototyping.
  3. Training Loss: The MLP model achieves a lower final training loss (0.0029) compared to the CNN model (0.0141), which might indicate potential overfitting to the training data.
  4. Model Size: Despite having a simpler structure, the MLP model has a larger file size (2.0MB) than the CNN model (177KB), indicating that MLPs require more parameters to achieve comparable performance.
  5. Overall Assessment: For the MNIST dataset, both models achieve high accuracy (>98%), with the CNN showing a slight edge. However, the MLP's faster training time makes it a viable option for this relatively simple image classification task.

This experiment validates CNN's advantages in image processing tasks while also demonstrating that for simpler datasets like MNIST, a well-designed MLP can achieve competitive results with faster training times.
